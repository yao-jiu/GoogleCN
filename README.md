\[2\]:**网站镜像**（mirror sites, mirrors；镜像站点，镜像网站）是指对网站内容的拷贝。镜像原意是光学里指的物体在镜面中所成之像。引申到计算机网络上，镜像通常用于为相同信息内容提供不同的源，特别是在下载量大的时候提供了一种可靠的网络连接。制作镜像是一种文件同步的过程。
